# Changelog — Mac Weird Keys

## v3.0 — 2026-03-15
- Windows keyboards: AZERTY FR, QWERTY, Arabic with AltGr/Shift+AltGr modifiers
- Mac/Windows OS toggle in Virtual Keyboard section
- Compare mode: overlay ghost labels to see layout differences
- Keyboard Detective: find-the-difference game between AZERTY and QWERTY
- Learn to Type: finger zones, finger finder, practice mode with streak counter
- Wrong Keyboard Simulator: type text and see what other layouts produce, or decode gibberish
- Advanced section groups non-essential features (Detective, Learn to Type, Simulator, Mac Tips)
- Layout reorganization: symbol grid always at top, bars below it

## v2.1 — 2026-03-15
- Virtual Keyboard: 3 full clickable keyboards (AZERTY, QWERTY, Arabic)
- Type by clicking keys, output to textarea, copy to clipboard
- Modifier keys (Shift, Option, Shift+Option) change key labels
- Arabic layout with RTL text direction
- Delete, Return, Tab, Space action keys

## v2.0 — 2026-03-15
- Floating Quick Bar: 10 most-needed symbols always visible
- Recently Copied: auto-saves last 10 copied symbols
- Search by description: type "email", "money", "code" to filter symbols
- Multi-copy: select multiple symbols and copy them all at once
- Favorite symbols: star your most-used, saved in localStorage
- Toast with paste hint: "Copied! Press Cmd+V to paste"
- Symbol of the Day: random daily symbol with fun fact
- Auto-detect layout: browser language sets default AZERTY/QWERTY

## v1.5 — 2026-03-15
- Quiz: Challenge mode with streak counter
- Cheat sheet: printable/copyable reference table

## v1.0 — 2026-03-15
- Initial release from Workshop-DIY template
- 65+ symbols across 7 categories
- Interactive keyboard with AZERTY/QWERTY toggle and X-Ray mode
- Mac Special Keys and Mac Secrets sections
- Symbol info cards with name, shortcut, analogy, copy button
- i18n (EN/FR/AR with RTL), 8 themes, PWA, sound effects, activity log
