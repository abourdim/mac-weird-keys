# Mac Weird Keys

**Find, learn & copy weird symbols you can't type on your Mac keyboard!**

A kid-friendly educational web app built with pure HTML, CSS & JavaScript. No frameworks. No dependencies.

---

## What it does

Kids (and adults!) struggle to type symbols like `@`, `{`, `[`, `|`, `~`, `\`, `#` on Mac keyboards — especially on AZERTY layouts. This app lets you:

- **Browse** a visual grid of 65+ symbols — grouped by category
- **Click** any symbol to copy it to clipboard instantly
- **Learn** the Mac shortcut to type each symbol (AZERTY & QWERTY)
- **Explore** interactive and virtual keyboards with X-Ray mode
- **Compare** Mac and Windows keyboard layouts side by side
- **Detect** differences between AZERTY and QWERTY in a detective game
- **Learn** touch typing with color-coded finger zones and practice mode
- **Simulate** what happens when you type on the wrong keyboard layout
- **Discover** Mac special keys and secret characters
- **Challenge** yourself with a point-and-click quiz
- **Print** a cheat sheet of all symbols & shortcuts

Everything is **point-and-click** — no typing required!

| Challenge Mode | Quiz with streak counter |
| Cheat Sheet | Printable/copyable reference table |
| Quick Bar | 10 most-needed symbols always visible |
| Favorites | Star your most-used symbols, saved in localStorage |
| Search | Filter symbols by description (email, money, code...) |

| 8 Themes | Mosque, Zellige, Andalus, Riad, Medina, Space, Jungle, Robot |
| PWA | Installable, works offline |
| Sound Effects | Click, success, error sounds |

## Section Layout

1. **Symbol Grid** — browse, search, filter, copy symbols
2. **Interactive Keyboard** — AZERTY/QWERTY with X-Ray mode
3. **Virtual Keyboard** — Mac & Windows layouts, compare mode
4. **Advanced** (collapsible)
   - Keyboard Detective — find differences between layouts
   - Learn to Type — finger zones, finder, practice
   - Wrong Keyboard Simulator — simulate & decode
   - Mac Tips — special keys & secrets
5. **Challenge Mode** — symbol quiz
6. **Cheat Sheet** — printable reference
